asciify-string
==============

Convert a unicode string to a readable ascii-only string.

## Usage


    var asciify = require('asciify-string')
    
    console.log(asciify('Déjà Vu')); // Deja Vu

## Asciify on NPM
You can install asciify-string via NPM, like so:

  npm install asciify-string
